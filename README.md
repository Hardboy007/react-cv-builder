# 📄 CV Application

A simple and responsive CV (Resume) builder built using React.

## 🚀 Features

* Add General Information (Name, Email, Phone)
* Add Education Details
* Add Work Experience
* Edit and update information anytime
* Dynamic reusable form component
* Clean and modern UI using Tailwind CSS

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* JavaScript (ES6)

## 📂 Project Structure

```
src/
 ├── components/
 │    ├── FormSection.jsx
 ├── App.jsx
```

## 💡 What I Learned

* React state management using `useState`
* Controlled components (forms)
* Conditional rendering
* Creating reusable components
* Handling dynamic form inputs
